Guide

          
Ce que fait chaque module, et le lien avec le programme BTS SISR.

        

      


      

        
Subnetting / VLSM

        
Calcule le plan d'adressage d'un réseau (adresse réseau, masque, broadcast, plage utilisable) à partir d'une IP et d'un CIDR. Le découpage VLSM répartit automatiquement un réseau de base en plusieurs sous-réseaux selon des besoins en hôtes différents — l'exercice classique d'adressage en SISR1.


        
VLAN

        
Génère une configuration de switch Cisco IOS complète : VLANs, ports en accès ou en trunk, SVI (interface virtuelle pour le routage inter-VLAN), VLAN voix, port-security. Base de tout TP de commutation.


        
Firewall / ACL

        
Génère des règles de filtrage sous deux formats : script iptables (serveur Linux) ou ACL Cisco (routeur). Inclut un testeur de règles pour vérifier quelle règle s'applique en premier à un paquet donné — utile pour réviser la logique des ACL avant un examen.


        
Topologie

        
Le module central : déclare plusieurs équipements (switchs, routeurs, PC, serveurs), les relie visuellement, et configure chacun individuellement. Couvre le routage inter-VLAN (router-on-a-stick), le DHCP, l'OSPF (zone unique), le NAT/PAT, l'EtherChannel et HSRP/VRRP — l'essentiel du programme SISR2 en administration réseau.


        
DNS

        
Génère un fichier de zone BIND (SOA, NS, A, CNAME, MX, PTR, TXT) — pour les TP de mise en place d'un serveur DNS.


        
Sauvegarde, export, rapport

        
Le travail dans Topologie, Firewall et DNS est sauvegardé automatiquement en local (rien n'est envoyé sur internet). Le bouton "Exporter" en bas de la sidebar télécharge tout le projet en JSON pour le sauvegarder ailleurs ou le transférer sur un autre PC. Le bouton "Rapport" génère un document imprimable (PDF via le navigateur) récapitulant toute la configuration — pratique pour un rendu de TP.


        
Utilisation hors-ligne

        
NetForge est une PWA : une fois ouverte une première fois, elle continue de fonctionner sans connexion internet (sauf l'export ZIP multi-équipements, qui a besoin de charger une librairie externe).
